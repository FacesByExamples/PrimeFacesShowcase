Output Panel

Output panel is a container element with several use cases, this example uses outputPanel to update a component page which doesn't exist on page initially and rendered based on a condition.

Source

<h:form prependId="false">
	<p:commandButton value="Show" update="campnou">
		<f:setPropertyActionListener value="\#{true}" target="\#{requestScope.renderCampNou}" />
	</p:commandButton>
	
	<p:outputPanel id="campnou">
		<p:graphicImage value="barca/campnou.jpg" rendered="\#{requestScope.renderCampNou}"/>
	</p:outputPanel>
</h:form>
