Tooltip - Global

In global mode, tooltip uses title values of JSF component, this is very efficient because placing one tooltip is enough instead of using a tooltip for each target component.

Source

<p:tooltip global="true"/>

<h:panelGrid columns="1">
			
	<h:outputText value="Text" title="Text to Enter" />
	
	<h:commandButton value="Submit" title="Save" />
	
	<h:outputLink value="#" title="Go to">
		<h:outputText value="Sample Link" />
	</h:outputLink>
	<h:inputText title="Input" />
	
</h:panelGrid>
