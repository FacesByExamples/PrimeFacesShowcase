Tooltip - Global

In global mode, tooltip uses title values of JSF components, this is very efficient because placing one tooltip is enough instead of using a tooltip for each target component.

Source

<p:tooltip global="true"/>

<h:panelGrid columns="1" cellpadding="5">

    <h:outputText value="Text" title="Tip on sample text" />

    <p:commandButton value="Submit" title="Click to save" type="button" />

    <h:outputLink value="#" title="Go to">
        <h:outputText value="Link" />
    </h:outputLink>

    <p:inputText title="This is a required field" />

</h:panelGrid>
