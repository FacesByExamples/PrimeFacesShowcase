Layout - Element

Layouts can also be created at element level. Example below is a simple split panel implementation.

Left Pane Center Pane

Source

<p:layout style="width:400px;height:200px;">
	<p:layoutUnit position="left" width="100" resizable="true" minWidth="50" maxWidth="200">
		Left Pane
	</p:layoutUnit>
    			
	<p:layoutUnit position="center">
		Center Pane
	</p:layoutUnit>
</p:layout>
