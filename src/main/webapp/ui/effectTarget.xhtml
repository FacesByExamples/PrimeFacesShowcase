Effect Target

By default, effects apply to their parents, this can be tuned with the for attribute so that effects are applied to another component.


Source

<h:outputLink id="lnk" value="#">
	<h:outputText value="Show the Barca Temple" />
	<p:effect type="puff" event="click" for="img">
		<f:param name="mode" value="'show'" />
	</p:effect>
